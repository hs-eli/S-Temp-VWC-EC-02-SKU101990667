This is an example of how to use soil moisture sensor S-Temp-VWC-EC-02.

This sensor works on MODBUS-RTU RS485 protocol. To inteface it with a PC you need an RS485-USB adapter. I use USB2-H-5002-M adapter.

The demo software is written in Python. It requires minimalmodbus python package to work.
